## 🌍 Visão Geral

Este projeto vai além de um simples portfólio.

Foi desenvolvido como uma experiência digital envolvente, com o objetivo de transmitir a identidade visual do estúdio e criar conexão com o usuário desde o primeiro contato.

A aplicação combina design moderno, animações sutis e navegação fluida para entregar uma experiência cinematográfica dentro do ambiente web.

---

## 🧱 Arquitetura

A aplicação segue uma estrutura moderna e escalável:

- Componentização reutilizável com React
- Tipagem estática com TypeScript
- Separação entre layout, lógica e estilo
- Organização voltada para performance e manutenção

---

## ⚡ Stack Tecnológica

- **Next.js** → Performance e renderização otimizada
- **React** → Estrutura baseada em componentes
- **TypeScript** → Segurança e escalabilidade
- **JavaScript (ES6+)** → Lógica da aplicação
- **TailwindCSS + CSS3** → Estilização moderna e responsiva
- **Figma** → Prototipação e design UI/UX

---

## 🎯 Diferenciais do Projeto

- 🎬 Experiência visual inspirada em cinema
- 🧠 Interações pensadas para engajamento do usuário
- 📱 Responsividade total em todos os dispositivos
- ⚡ Navegação fluida e intuitiva
- 🎨 Design moderno e imersivo

---

## 📞 Integrações

O projeto conta com canais diretos de conversão:

- 📲 Redirecionamento para **WhatsApp**
- 📸 Integração com **Instagram**
- 💬 Facilidade de contato com o estúdio em poucos cliques

---

## 🛠 Desafios Técnicos

Durante o desenvolvimento, o foco foi equilibrar:

- Performance vs Animações
- Design imersivo vs Usabilidade
- Estética visual vs Responsividade

O resultado foi uma aplicação leve, fluida e altamente adaptável.
